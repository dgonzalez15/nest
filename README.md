## Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar
```bash
$ yarn install
```
3. Tener Nest CLI instalado
```
npm i -g @nestjs/cli
```
4. Levantar la base de datos
```
docker-compose up -d
```
## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod

# instalar axios
$ yarn add axios@0.27.2
```
